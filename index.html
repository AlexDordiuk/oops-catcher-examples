Basic Usage

import Oops from 'oops-catcher'

// Initialize
Oops.init({
  endpoint: '/api/feedback',
  captureConsole: true
})

// Report a bug
Oops.report({
  message: "Button doesn't work",
  category: 'bug',
  severity: 'high'
})

A lightweight feedback collector that captures console logs, user agent info, and user reports.
